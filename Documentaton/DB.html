NAME

PocketNews::DB

SYNOPSIS


    use PocketNews::DB;
    ...
    my $db = PocketNews::DB->new(
                _filename =>'foo.sqlite',
            );

DESCRIPTION

DB wrapper

METHODS

new

Usage
        my $db = PocketNews::DB->new(
                    _filename =>"foo.sqlite",
                );

exists

Description

Check if this article exists in the database.

Usage

$object->exists("title of the article");

Return

Returns the count of the fields with such title.

addNew

Description

Add new article to the database.

Usage

$object->addNew("title of the article");

Return

return 1 on succes, -1 on error;

restoreAI

Description

Subroutine for restorin the Auto Increment value to 0 for table.

Usage

$db->restoreAI("Table");

Return

Returns the number of affected rows 1 on succes otherwise 0.

getLastId

Description

Subroutine for getting the last Auto Increment value for table;

Usage

$db->getLastId("Table");

Return

Returns scalar - the last inserted id.
